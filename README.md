# AiiDA tutorials

This repository contains a collection of texts used for
various AiiDA tutorials.

Visit http://aiidateam.github.io/aiida-tutorials

## Building locally

```
cd docs
pip install -r requirements.txt
make
# open build/html/index.html
```
